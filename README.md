## Demo
[https://ethereal-node-api.vercel.app/api/create-email](https://ethereal-node-api.vercel.app/api/create)

## 1. Install dependencies
```
  yarn or npm install
```

## 1.2 Configure enviroments
```
  cp .env.sample .env
```

## 1.3 Run project
```
  yarn dev
```

## 1.4 Send GET method
```
http://localhost:3000/api/create-email
```
